# 07. RAG 핸즈온 — HR 지식봇 (사내규정 챗봇)

**Bedrock Knowledge Bases**를 AWS 콘솔에서 구성해, 사내 규정 문서를 근거로 답하는 **RAG(Retrieval-Augmented Generation) 챗봇**을 만든 핸즈온.

## RAG 흐름

```
사내규정 문서(md) → S3 업로드 → Knowledge Base (청킹 + 임베딩 + 벡터 스토어)
→ 질문 → 관련 청크 검색(Retrieve) → LLM이 근거 기반 답변 생성(Generate)
```
